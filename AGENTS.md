# AGENTS.md – Strana PLÁN

## Projekt

Oficiální web politické strany PLÁN. Pravicová strana s heslem „Pravicová prosperita pro moderní budoucnost."

## Konvence

- Jazyk webu: čeština
- Barvy: modrá (#004B93), červená (#B22234), bílá (#FFFFFF)
- Styl: moderní, čistý, profesionální
- Technologie: vanilla HTML5 + CSS3, žádné frameworky
- Všechna práva vyhrazena – nekopírovat obsah

## Struktura menu

Home | Priority | Program | Kontakt | Připoj se

- Kontaktní údaje budou doplněny později
