# Pravidla diskuze

## Pravidla chování

1. **Respektujte ostatní** – žádné urážky, vyhrožování ani diskriminace
2. **Pište věcně** – držte se tématu, žádný spam ani reklama
3. **Žádné nelegální obsahy** – podněcování k násilí, nenávistné projevy apod.
4. **Žádné osobní údaje** – nezveřejňujte osobní údaje jiných osob

## Důsledky porušení

- Příspěvek bude odstraněn
- Opakované porušení = zablokování uživatele
